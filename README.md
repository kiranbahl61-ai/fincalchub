# FinCalcHub

A modern financial hub website providing calculators, educational content, investment instruments, glossary, and growth strategies.

## Features

- Financial Calculators: Loan, SIP, Retirement, EMI, Tax, Investment Growth, Goal Planning, Inflation, CAGR, PPF, EPF
- Personal Finance GPS™: health score, protection review, tax optimization, diversification insights, roadmap
- Educational Content: Budgeting, saving, debt management, wealth building
- Investment Instruments: Stocks, Mutual Funds, Insurance, Bonds, ETFs, Alternatives
- Glossary: Searchable financial terms
- Strategies: Asset allocation, risk management, compounding

## Disclaimer

This site is for educational purposes only and not financial advice.

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Tech Stack

- React
- TypeScript
- Vite
- React Router
